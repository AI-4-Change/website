Spring Program

Spring Program was a 2 day event with interactive workshops and networking sessions for students interested in learning about Artificial Intelligence. The event featured a broad range of topics such as Introductory and Advanced Python and applying AI in various fields. Not only did these workshops help students learn and gain practical experience, but they also allowed them to network with other students attending the event.

Topics Covered

The Spring Program Covered many topics related to AI and computer science.

Some topics include:

  • Introduction to Linear Regression
  • Paradigms of AI
  • Introduction to K-Nearest Neighbors
  • Introduction to Support Vector Machines
  • Intro to NLP
  • Introduction to Python
  • Advanced Python

Reach

The Spring Program reached students from accross the world in places such as India, Brazil, and Indonasia.
